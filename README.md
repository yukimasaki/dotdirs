# 何ができるの？

- mcp.jsonの自動生成
- ルールファイル、ワークフローファイルの一元管理（Cursor/Antigravity対応）

# 使い方

1. プロジェクトのルートディレクトリに移動する

```bash
cd path/to/your/project
```

2. インストールスクリプトを実行する

```bash
curl -fsSL https://raw.githubusercontent.com/yukimasaki/dotdirs/main/init.sh | bash -s -- .
```
